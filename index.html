Що таке Flex(box)?

Flexbox (Flexible Box Layout) — це сучасний метод розташування елементів у вебдизайні. Він дозволяє розміщувати елементи гнучко та адаптивно у контейнерах, керуючи вирівнюванням, розподіленням простору та адаптацією до різних розмірів екрану.

Основні властивості Flexbox

display: flex

Встановлює контейнер як flex-контейнер.

flex-direction

Визначає напрямок вирівнювання елементів: рядок або стовпець.

justify-content

Вирівнює елементи по головній осі.

align-items

Вирівнює елементи по поперечній осі.

flex-wrap

Дозволяє елементам переноситися на новий рядок.

Порівняння Flexbox з Float

До появи Flexbox для вирівнювання елементів використовували float. Проте Flexbox надає більшу гнучкість та контроль над розташуванням елементів, дозволяючи простіше вирівнювати блоки по центру та працювати з адаптивними макетами.
